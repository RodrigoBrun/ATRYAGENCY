# ATRY — Phosphor Icons global

Este parche implementa Phosphor Icons en toda la web usando archivos locales.

## Archivos modificados

- index.html
- proyectos.html
- servicios.html
- estudio.html
- playground.html
- styles.css
- script.js

## Archivos nuevos

```text
assets/
└── icons/
    └── phosphor/
        ├── regular/
        │   ├── style.css
        │   └── Phosphor.woff2
        ├── bold/
        │   ├── style.css
        │   └── Phosphor-Bold.woff2
        └── LICENSE.txt
```

## Qué se reemplazó

- Flechas de botones principales.
- Iconos de abrir sitio.
- Selector Desktop / Tablet / Mobile.
- Botones Ver caso.
- Botones de acordeón de servicios.
- Cierre de modales.
- Envío del formulario.
- Volver arriba.
- CTA generales.
- Iconos ya existentes del Playground apuntan ahora a la carpeta global.

## Instalación

Copiar todos los archivos del parche respetando la estructura.

No depende de CDN y funciona en GitHub Pages.
